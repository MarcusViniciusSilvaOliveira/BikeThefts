Importante:
- Navegadores homologados: Firefox 84.0, MicrosoftEgde 87.0
- Necessita do Node.js instalado na máquina.

Para execução do projeto:
- Abra o prompt de comando.
- No CMD, acesse o diretorio da pasta do projeto "bikethefts".
- Digite o comando "npm -i" sem as aspas e espere o processo concluir.
- Digite o comando "npm start" sem as aspas e espere o processo concluir.
- Abra um navegador de sua preferência que esteja entre os navegadores homologados.
